## 🎮 전략적 깊이

### 적 AI의 선택
1. **안전 플레이**: Slam, FrostTouch 등 → 약점 없음
2. **공격적 플레이**: HeavySlam, FireSpell 등 → 큰 데미지 but 약점 노출
3. **버프 플레이**: RageBuff, StormCharge → 다음 턴 강화

### 플레이어의 대응
1. **속성 확인**: 적이 Fire 속성 → IceSpell 사용 (1.5배!)
2. **텔레그래프 반응**: HeavySlam 예고 → Defend() (50% 감소)
3. **패턴 학습**: `EnemyLastAction(FireSpell)` → 3턴간 Ice 약점!

### 예시 시나리오
```
턴 1: 적 Neutral → HeavySlam 사용
      → Fire 속성 획득 (3턴)
      → 플레이어: "적이 Fire다! IceSpell!"

턴 2: 적 Fire (2턴 남음) → Slam 사용 (안전)
      → 플레이어: IceSpell (1.5배 데미지!)

턴 3: 적 Fire (1턴 남음) → RageBuff
      → 플레이어: IceSpell (1.5배 데미지!)

턴 4: 적 Neutral로 복귀
      → 플레이어: "약점 사라짐, 일반 공격"
```

## 🎯 핵심 포인트

1. **강공격 = 속성 노출**: 큰 데미지 대신 3턴간 약점 생김
2. **속성 마법 = 속성 노출**: 속성 데미지 + CC 대신 약점 생김
3. **일반 공격 = 안전**: 약한 데미지지만 약점 없음
4. **특수 스킬 = 안전**: 버프/디버프는 속성 변경 없음
5. **3턴 지속**: 속성은 3턴간 유지, 이후 Neutral 복귀
